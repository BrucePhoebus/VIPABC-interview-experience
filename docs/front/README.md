# 前端规范概述

  规范的价值永远是约束我们的行为，让我们拥有更体系化的知识，同时也有团队风格。

  基于生产力的目的，让我们写的代码更有可读性、可维护。

## 前端都有哪些规范

#### 一、编程规范

* 作为一个开发人员，良好的编程风格明显可以提高我们的开发效率，写代码如穿衣，看人靠衣装，敲码靠规范

#### 二、eslint规范辅助

* eslint是很好规范我们编程细节的规范工具，配置好这个工具，能让我们的代码焕然一新
