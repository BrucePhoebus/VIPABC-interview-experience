# 面经工作学习笔记

## 使用

```BASH
# 拉去项目
git clone https://gitee.com/s_phoebus/vipabc-interview-experience
# 安装依赖
npm install # yarn
# 运行
npm run start
```

> [运行后浏览器访问](http://localhost:8888)

## 注意事项

#### 关于注释

建议使用VSCode，然后添加`koroFileHeader`，这样方便添加注释，`.vscode/settings.json`文件是本地化一些配置，每次需要修改就打开`.gitignore`对这个文件许可即可
